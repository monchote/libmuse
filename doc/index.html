Muse SDK Documentation

This library enables communication with and control of Muse headbands from client software.

Quick start

iOS

Every Objective-C class is prefixed with IXN. Start with IXNMuseManager and explore from there, or use the Classes tab.

Threading

Most libmuse methods are thread safe, except execute() and run_asynchronously(). Each method describes its threading concerns if there are any.

Exceptions

The libmuse native library catches and handles every exception that it can reasonably be expected to do so – e.g., reconnecting on stream corruption. There are, however, some situations in which native code can still throw, such as when it fails to create a Bluetooth socket. Any API method that can throw documents this.
