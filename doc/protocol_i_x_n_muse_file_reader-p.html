Detailed Description

Class allows to access data from 'muse' protobuf file, which could be written using MuseFileWriter or another file writer, which corresponds to Muse protobuf schema. When file reader is created it always starts reading from the beginning of the file to the end.
Threading: It is NOT thread safe, you have to provide your own synchronization mechanism if you plat to calls methods of this class from different threads.

Method Documentation

- (void) close

Closes the file. Calls Muse File close. You don't have to call close explicitly unless you want to close file immediately. close is called automatically, when file reader object is destroyed.

- (IXNAnnotationData *) getAnnotation

Returns annotation structure by parsing protobuf message under current pointer.
Throws exception if current message type is not annotation.

- (IXNComputingDeviceConfiguration *) getComputingDeviceConfiguration

Returns computing device configuration structure by parsing protobuf message under current pointer.
Throws exception if current message type is not computing device configuration.

- (IXNMuseConfiguration *) getConfiguration

Returns muse configuration structure by parsing protobuf message under current pointer.
Throws exception if current message type is not muse configuration.

- (IXNMuseDataPacket *) getDataPacket

Returns muse data packet by parsing protobuf message under current pointer. Use this method to get EEG, Accelerometer, Battery, Quantization and drooped packets.
Throws exception if current message type is not muse data packet.

- (IXNDspData *) getDsp

Returns muse configuration structure by parsing protobuf message under current pointer.
Throws exception if current message type is not muse configuration.

- (int32_t) getMessageId

Returns id of the message on current pointer position. Returns '-1' if config id isn't found in the protobuf file.

- (int64_t) getMessageTimestamp

Returns timestamp of the message on current pointer position.

- (IXNMessageType) getMessageType

Returns type of the message on current pointer position

- (IXNMuseVersion *) getVersion

Returns muse version structure by parsing protobuf message under current pointer.
Throws exception if current message type is not muse version.

- (BOOL) gotoNextMessage

Moves pointer to the next MuseData message in protobuf stream.
Returns true if successful, false - if failed (reached the end of stream/file).

- (void) open

Opens an existing file Use this method if you explicitly closed file and want to open it again. Calls Muse File open.
